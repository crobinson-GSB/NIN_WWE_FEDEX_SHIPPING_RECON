# GSB Digital — Shipping Reconciliation Tool

Upload your Vision export and vendor invoices to automatically match and flag shipping cost discrepancies.

## Supported Vendors
- NIN (Courier) — .xls invoice
- WWE / UPS — .xls invoice
- FedEx — coming soon

## How to use
1. Go to the app URL
2. Upload the Vision export (.txt or .xlsx)
3. Upload the NIN and WWE invoices
4. Click Run Reconciliation
5. Download the Excel report
